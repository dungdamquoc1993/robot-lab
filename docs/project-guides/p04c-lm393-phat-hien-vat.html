Chỉ thêm: module LM393 @ d20–d22 + 3 dây jumper mới (đỏ, cam thứ hai, nâu). Dây cam LED và dây cam module có thể cùng cắm lỗ a3 (GND chung) — không cần rút dây cam cũ.

An toàn

Cấp module bằng 3V3 (dây đỏ → a2). Đo OUT bằng đồng hồ nếu nghi ngờ >3.3V trước khi nối D19.

Cắm module — từng chân

Module đặt nửa trái, cột d, hàng 20–22 (xa LED ở hàng 27–30). Đối chiếu nhãn in trên module, không đoán thứ tự chân:

Nhãn trên moduleCắm vào breadboard
VCCd20
GNDd21
OUTd22

Nếu 3 chân module nằm ngang một hàng thay vì dọc — vẫn map đúng VCC/GND/OUT vào 3 lỗ tương ứng ở hàng 20–22; giữ chúng cùng nửa trái (a–e).

Các bước làm

  1. Giữ mạch LED P4; gỡ jumper nâu P3 nếu còn.
  2. Cắm module d20–d22 theo nhãn VCC/GND/OUT.
  3. Lần lượt 3 dây: đỏ a2→d20 · cam a3→d21 · nâu a11→d22.
  4. main.cpp: comment P4/P5, bỏ comment p04c_setup / p04c_loop.
  5. pio run -t upload · Serial Monitor 115200.
  6. Đưa tay trước module IR — chỉnh biến trở trên module nếu không đổi trạng thái.

Hoàn thành khi

  • Serial in OBSTACLE / clear khi có/không vật.
  • LED P4 sáng khi phát hiện vật (cùng D2).
  • Nếu logic ngược → đổi OBSTACLE_ACTIVE_LOW trong firmware.
