# P4 — Breathing LED

Dùng lại **mạch LED từ P2** trên D2. Nút P3 có thể gỡ hoặc để nguyên — firmware P4 không đọc D18.

## Linh kiện trên board

| Linh kiện | Tọa độ | Ghi chú |
|-----------|--------|---------|
| ESP32 3V3 | b2 | |
| ESP32 GND | b3 | |
| ESP32 D2 | b5 | PWM → LED |
| Điện trở | d27 ↔ h27 | Vắt rãnh, 220Ω |
| LED anode (+) | j27 | |
| LED cathode (−) | j30 | |

## Jumper

| Màu | Từ | Đến | Tín hiệu |
|-----|-----|-----|----------|
| Cam | a3 | i30 | GND → hàng 30 phải (LED−) |
| Vàng | a5 | a27 | D2 → mạch LED |

## Atomic path — PWM duty tăng → LED sáng dần

1. `loop()` tăng `duty` 0→255, mỗi bước `ledcWrite(2, duty)` + `delay(8)`.
2. `ledcWrite` — driver LEDC bật/tắt GPIO2 @ b5 với duty cycle tương ứng.
3. Chu kỳ PWM ~5kHz — mắt thấy độ sáng trung bình thay vì nhấp nháy.
4. Dòng trung bình: 3.3V × duty% → b5 → vàng → a27 → d27 → R → h27 → j27 → LED → j30 → cam → GND.

## Firmware ↔ hardware

| Code | GPIO | Coord | Khi chạy |
|------|------|-------|----------|
| `LED_PIN` | 2 | b5 | PWM 5kHz, 8-bit (0–255) |
| `PWM_FREQ` | — | — | 5000 Hz |
| `FADE_STEP_MS` | — | — | 8ms mỗi bước duty |

## Gotchas

- Vẫn cần **điện trở nối tiếp** — PWM không thay thế giới hạn dòng.
- Nếu giữ nút P3 trên board: không nối D18 xuống GND nhầm hàng (xem [p03.md](p03.md)).
