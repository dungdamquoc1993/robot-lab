# P8 — Web server, breadboard trắng

P8 mặc định dùng **ESP32 rời breadboard**: cắm USB vào máy tính, ESP32 tự phát Wi-Fi AP `RobotLab-P8`, điện thoại mở web page để bật/tắt onboard LED GPIO2.

Breadboard để trắng hoàn toàn. Không cắm ESP32 lên MB102, không nối rail nguồn, không nối LED ngoài.

## Linh kiện trên board

| Linh kiện | Tọa độ | Ghi chú |
|-----------|--------|---------|
| Breadboard MB102 | trống | Không cắm dây hoặc linh kiện |
| ESP32 DevKit V1 | off-board | Cấp nguồn qua USB |
| Onboard LED | trên ESP32 | GPIO2, không có tọa độ breadboard |
| Servo tùy chọn | off-board | Chỉ dùng sau khi P8 web server chạy ổn |

## Kết nối mặc định

| Màu gợi ý | Từ | Đến | Tín hiệu |
|-----------|-----|-----|----------|
| — | USB máy tính | ESP32 micro-USB | 5V USB + Serial |
| — | ESP32 GPIO2 | LED onboard | Output 3.3V nội bộ trên board |

## Servo tùy chọn sau

Nếu muốn thử slider servo trên web page, dùng lại nguyên tắc P7/P7b:

| Servo SG90 | Nối tới | Ghi chú |
|------------|---------|---------|
| Signal vàng/cam | ESP32 D18 | GPIO18 chỉ là signal 3.3V |
| Đỏ | Nguồn 5V riêng | Không lấy từ chân 3V3 |
| Nâu | GND nguồn 5V riêng + GND ESP32 | Bắt buộc chung mass |

Không cấp servo từ rail 3.3V của ESP32. Nếu servo rung mạnh, nóng nhanh, hoặc ESP32 reset: ngắt 5V servo ngay.

## Atomic path — onboard LED

1. Điện thoại gửi HTTP request tới `http://192.168.4.1/api/led?state=toggle`.
2. `WebServer` trên ESP32 gọi `p08HandleLed()`.
3. `p08WriteLed()` đổi biến `p08LedOn` và gọi `digitalWrite(2, HIGH/LOW)`.
4. GPIO2 trên ESP32 đổi mức logic 3.3V.
5. LED onboard sáng/tắt ngay trên board ESP32; breadboard không có dòng điện nào đi qua.

## Firmware ↔ hardware

| Code | GPIO | Coord | Khi chạy |
|------|------|-------|----------|
| `ONBOARD_LED_PIN` | 2 | onboard ESP32 | LED xanh trên ESP32 bật/tắt qua web |
| `OPTIONAL_SERVO_PIN` | 18 | off-board | Signal servo tùy chọn; chỉ phát PWM khi dùng slider |
| `AP_SSID` | — | — | Wi-Fi AP `RobotLab-P8` |

## Gotchas

- Điện thoại cần kết nối Wi-Fi `RobotLab-P8`, mật khẩu `robotlab8`, rồi mở `http://192.168.4.1`.
- AP này không có Internet; điện thoại có thể hiện cảnh báo "No Internet", cứ giữ kết nối.
- GPIO ESP32 là logic 3.3V; không đưa 5V vào GPIO.
- Nếu chuyển sang Wi-Fi nhà, có thể build với `P08_WIFI_SSID` và `P08_WIFI_PASSWORD`; không commit mật khẩu thật vào repo.
