# P9 — OLED I2C + LDR, ESP32 rời breadboard

P9 bắt đầu từ **breadboard trắng**. ESP32 không cắm lên breadboard; dùng dây Đực-Cái từ chân ESP32 rời tới các hàng landing trên MB102.

Mục tiêu:

- OLED 0.96" hiển thị text qua I2C.
- LDR module đưa AO vào ADC D34.
- Serial vẫn log được nếu OLED chưa nhận.

## Linh kiện trên board

| Linh kiện | Tọa độ | Ghi chú |
|-----------|--------|---------|
| OLED GND | e1 | Đối chiếu nhãn trên module: `GND VDD SCK SDA` |
| OLED VDD | e2 | Cấp 3V3 |
| OLED SCK | e3 | I2C clock từ ESP32 D22 |
| OLED SDA | e4 | I2C data từ ESP32 D21 |
| LDR VCC | e20 | Cấp 3V3 |
| LDR GND | e21 | GND chung |
| LDR DO | — | Không nối trong P9 |
| LDR AO | e23 | Analog output vào ESP32 D34 |
| ESP32 | off-board | Cấp nguồn qua USB; jumper cắm trực tiếp vào chân ESP32 |

## Jumper

**Cùng hàng nửa trái (a-e) = cùng node**. Module cắm cột `e`, jumper cắm cột `a` để không tranh lỗ.

| Màu gợi ý | Từ | Đến | Tín hiệu |
|-----------|-----|-----|----------|
| Nâu | OLED GND | e1 | OLED GND landing |
| Đỏ | OLED VDD | e2 | OLED 3V3 landing |
| Blue | OLED SCK | e3 | OLED clock landing |
| Tím | OLED SDA | e4 | OLED data landing |
| Đen | ESP32 GND | a1 | GND → OLED GND @ e1 |
| Cam | ESP32 3V3 | a2 | 3V3 → OLED VDD @ e2 |
| Xanh ngọc | ESP32 D22 | a3 | SCK/SCL → OLED SCK @ e3 |
| Ghi | ESP32 D21 | a4 | SDA → OLED SDA @ e4 |
| Cam2 đực-đực | b2 | a20 | Chia 3V3 từ hàng 2 sang LDR VCC @ e20 |
| Trắng | ESP32 GND | a21 | GND → LDR GND @ e21 |
| Vàng | ESP32 D34 | a23 | ADC ← LDR AO @ e23 |

Không nối LDR `DO`.

## Atomic path — sensor lên màn hình

1. LDR đổi điện trở theo ánh sáng; module tạo điện áp analog tại `AO`.
2. `analogRead(34)` đọc điện áp đó thành số 0-4095.
3. Firmware đổi số ADC thành phần trăm gần đúng.
4. ESP32 D21/D22 gửi byte qua bus I2C: D21 = SDA dữ liệu, D22 = SCL nhịp clock.
5. OLED nhận frame buffer và tự phát sáng pixel để hiển thị ADC/%.

## Firmware ↔ hardware

| Code | GPIO | Coord | Khi chạy |
|------|------|-------|----------|
| `I2C_SDA_PIN` | 21 | off-board → a4/e4 | OLED SDA |
| `I2C_SCL_PIN` | 22 | off-board → a3/e3 | OLED SCK |
| `LDR_AO_PIN` | 34 | off-board → a23/e23 | ADC đọc ánh sáng |
| OLED address | 0x3C hoặc 0x3D | I2C bus | Auto-scan khi boot |

## Gotchas

- Cấp OLED và LDR bằng **3V3**, không đưa 5V vào ESP32 GPIO.
- Thứ tự chân module OLED của bạn là `GND VDD SCK SDA`; `VDD` = nguồn 3V3, `SCK` = I2C clock/SCL.
- Nếu OLED không sáng, mở Serial Monitor 115200; firmware sẽ in I2C scan.
- D21/D22 là default I2C phổ biến trên ESP32, nhưng ở bài này code vẫn gọi rõ `Wire.begin(21, 22)`.
