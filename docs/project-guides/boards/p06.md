# P6 — Cảm biến ánh sáng (module LDR)

Đọc chân **AO** (analog) của module quang trở qua **D34** (ADC). Chân **DO** không dùng trong bài cơ bản.

**Gỡ biến trở P5** (nếu còn) — thay bằng module LDR cùng vùng hàng 20–22.

**Cột phải ESP32:** chân board nằm @ **j** (không cắm dây vào **i** — bị board che). Jumper tới D34/GND/VIN: **j13**, **j3**, **j2**, …

## Linh kiện trên board

| Linh kiện | Tọa độ | Ghi chú |
|-----------|--------|---------|
| ESP32 3V3 | b2 | Jumper đỏ → a2 |
| ESP32 GND | b3 | Jumper cam → a3 |
| ESP32 D34 | j13 | ADC ← AO module |
| Module VCC | d20 | Nhãn trên PCB |
| Module GND | d21 | |
| Module AO | d24 | 4 chân thưa — DO thường @ d22 hoặc d23, không nối |
| Module DO | — | Không nối (tùy chọn D19 sau) |

## Jumper (3 sợi)

**Cùng hàng nửa trái (a–e) = cùng node** — module cắm **d**, dây cắm **a** (không share lỗ với chân module).

| Màu gợi ý | Từ | Đến | Tín hiệu |
|-----------|-----|-----|----------|
| Đỏ | a2 | **a20** | 3V3 → VCC @ d20 |
| Cam | a3 | **a21** | GND → GND @ d21 |
| Xanh | **a24** | j13 | AO @ d24 → D34 (ADC) |

## Atomic path — ánh sáng đổi → số ADC đổi

1. LDR trên module đổi điện trở theo ánh sáng → divider trên PCB đổi điện áp **AO**.
2. `analogRead(34)` @ j13 — ADC 12-bit ≈ 0–4095.
3. Sáng hơn hoặc tối hơn (tùy module) → `adc` tăng/giảm — ghi min/max khi che tay / đèn bàn.

## Firmware ↔ hardware

| Code | GPIO | Coord | Khi chạy |
|------|------|-------|----------|
| `LDR_AO_PIN` | 34 | j13 | ADC input-only |

## Gotchas

- Cấp module **3V3** (a2), không 5V — AO không vượt 3.3V.
- Đối chiếu nhãn **VCC · GND · DO · AO** trên PCB — thứ tự chân có thể khác LM393.
- Xoay biến trở **trên module** chỉ ảnh hưởng ngưỡng **DO**, không ảnh hưởng AO.
- D34 @ **j13** — cột **j** là lỗ trống cạnh chân ESP32; không tranh lỗ với board.
