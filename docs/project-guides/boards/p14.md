# P14 — Relay 2 kênh 5V, ESP32 rời breadboard

P14 bắt đầu từ **breadboard trắng**. ESP32 và relay module đều để rời breadboard; MB102 chỉ làm landing cho jumper.

Mục tiêu:

- ESP32 điều khiển `IN1` và `IN2` của module relay 2 kênh.
- Nghe relay K1/K2 đóng/ngắt “tạch”.
- Chưa đấu tải vào cọc vít xanh, chưa dùng điện lưới AC.

## Linh kiện trên board

| Linh kiện | Tọa độ | Ghi chú |
|-----------|--------|---------|
| Relay module GND | e20 | Header điều khiển `GND IN1 IN2 VCC` |
| Relay module IN1 | e21 | Kênh relay 1 |
| Relay module IN2 | e22 | Kênh relay 2 |
| Relay module VCC | e23 | Cấp 5V từ ESP32 VIN/5V qua USB |
| ESP32 | off-board | Cấp nguồn qua USB; jumper cắm trực tiếp vào chân ESP32 |

Không cắm gì vào cọc vít xanh `NO/COM/NC` trong bài test này.

## Jumper

**Cùng hàng nửa trái (a-e) = cùng node**. Module cắm/câu tới cột `e`, ESP32 landing ở cột `a`.

| Màu gợi ý | Từ | Đến | Tín hiệu |
|-----------|-----|-----|----------|
| Đen | ESP32 GND | a20 | GND chung với relay |
| Xanh lá | ESP32 D26 | a21 | Điều khiển relay K1 / IN1 |
| Tím | ESP32 D27 | a22 | Điều khiển relay K2 / IN2 |
| Đỏ | ESP32 VIN/5V | a23 | Nguồn 5V cho relay module |
| Đen relay | Relay GND | e20 | Header relay `GND` |
| Xanh relay | Relay IN1 | e21 | Header relay `IN1` |
| Tím relay | Relay IN2 | e22 | Header relay `IN2` |
| Đỏ relay | Relay VCC | e23 | Header relay `VCC` |

Giữ nguyên jumper cap nhỏ trên module nếu nó đang nối `JD-VCC` và `VCC`. Không nối header `JD-VCC/VCC/GND` riêng ở P14 basic.

## Atomic path — relay click

1. Firmware gọi `p14SetRelay(26, true)`.
2. GPIO26 chuyển sang `OUTPUT LOW`, giống một công tắc kéo `IN1` xuống GND.
3. Dòng điều khiển đi từ relay `VCC` 5V qua mạch input/opto của module về `IN1`.
4. `IN1` về ESP32 D26 qua `e21/a21`, rồi D26 sink xuống GND.
5. Module kích transistor/coil relay K1, relay phát tiếng “tạch”.
6. Khi OFF, firmware đặt D26 về `INPUT`; ESP32 thả dây `IN1`, module tự nhả relay.

## Firmware ↔ hardware

| Code | GPIO | Coord | Khi chạy |
|------|------|-------|----------|
| `RELAY1_PIN` | 26 | off-board → a21/e21 | Kéo LOW để bật K1 |
| `RELAY2_PIN` | 27 | off-board → a22/e22 | Kéo LOW để bật K2 |
| Relay `VCC` | VIN/5V | off-board → a23/e23 | Nguồn 5V cho module |
| Relay `GND` | GND | off-board → a20/e20 | GND chung |

## Gotchas

- ESP32 GPIO là **3.3V logic**; không đưa 5V trực tiếp vào GPIO.
- Code P14 không drive GPIO HIGH; OFF là `INPUT`, ON là `OUTPUT LOW`, hợp với relay module active-LOW.
- Chỉ test tiếng relay và LED onboard module. Không đấu điện lưới AC/220V vào cọc vít xanh.
- Nếu relay không “tạch”, kiểm tra thứ tự header relay đúng là `GND IN1 IN2 VCC`.
