# P5 — Biến trở → LED

Giữ **mạch LED P2/P4** trên D2. Thêm **biến trở 10K** làm voltage divider — chân giữa vào **D34** (ADC).

**Nhận diện:** ảnh `photos/bien-tro-10k.jpg` — 3 chân + trục xoay (trong túi `nut-bam-12x12-bien-tro-10k.jpg`, không nhầm với nút vuông 4 chân hay buzzer 2 chân).

## Linh kiện trên board

| Linh kiện | Tọa độ | Ghi chú |
|-----------|--------|---------|
| ESP32 3V3 | b2 | |
| ESP32 GND | b3 | |
| ESP32 D2 | b5 | PWM → LED |
| ESP32 D34 | j13 | ADC ← biến trở chân giữa |
| Biến trở 10K chân 1 | d20 | Nối 3V3 |
| Biến trở chân giữa | d22 | Wiper → D34 (chân thưa: có thể d21) |
| Biến trở chân 3 | d24 | Nối GND (chân thưa: có thể d22) |
| Điện trở | d27 ↔ h27 | 220Ω, vắt rãnh |
| LED anode (+) | j27 | |
| LED cathode (−) | j30 | |

## Jumper

| Màu | Từ | Đến | Tín hiệu |
|-----|-----|-----|----------|
| Xanh | a22 | j13 | Wiper → D34 (ADC) |
| Đỏ | a2 | a20 | 3V3 → biến trở chân 1 |
| Cam | a3 | a24 | GND → biến trở chân 3 |
| Vàng | a5 | a27 | D2 → mạch LED |
| Cam | a3 | j30 | GND → LED− |

## Atomic path — xoay biến trở → LED đổi sáng

1. `analogRead(34)` — ADC đo điện áp chân giữa biến trở (0–3.3V ≈ 0–4095).
2. Xoay núm → tỉ lệ R trên/dưới wiper đổi → điện áp @ d21 thay đổi.
3. `map(adc, 0, 4095, 0, 255)` → duty PWM.
4. `ledcWrite` — LEDC phát PWM @ b5 theo duty.
5. Dòng LED: 3.3V × duty% → b5 → vàng → … → LED → GND.

## Firmware ↔ hardware

| Code | GPIO | Coord | Khi chạy |
|------|------|-------|----------|
| `POT_PIN` | 34 | j13 | ADC input-only, đọc biến trở |
| `LED_PIN` | 2 | b5 | PWM output LED |
| `map(...)` | — | — | ADC 12-bit → duty 8-bit |

## Gotchas

- **GPIO 34 chỉ input** — không `digitalWrite` được.
- Chỉ cấp **3.3V** cho biến trở, không 5V (ADC max 3.3V).
- Xoay ngược ý → đổi dây 3V3/GND hai đầu biến trở (wiper giữ nguyên @ D34).
- Gỡ nút P3 / jumper D18 nếu không dùng.
