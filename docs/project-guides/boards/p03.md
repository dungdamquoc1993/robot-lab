# P3 — Nút bật/tắt LED

Trạng thái board **đã chạy ổn** (2026-07-04) — D18 trên **hàng 33 trái**, không hàng 30 trái.

## Linh kiện trên board

| Linh kiện | Tọa độ | Ghi chú |
|-----------|--------|---------|
| ESP32 3V3 | b2 | |
| ESP32 GND | b3 | |
| ESP32 D2 | b5 | LED output |
| ESP32 D18 | b10 | Nút input |
| Điện trở | d27 ↔ h27 | Vắt rãnh |
| LED anode (+) | j27 | |
| LED cathode (−) | j30 | |
| Nút chân 1 | d30 | Hàng 30 trái |
| Nút chân 2 | g30 | Hàng 30 phải — cực A |
| Nút chân 3 | d33 | Hàng 33 trái — cực B, **nâu/D18** |
| Nút chân 4 | g33 | Hàng 33 phải |

## Jumper

| Màu | Từ | Đến | Tín hiệu |
|-----|-----|-----|----------|
| Cam | a3 | i30 | GND → hàng 30 phải (LED−, nút cực 2) |
| Vàng | a5 | a27 | D2 → mạch LED |
| Nâu | a10 | a33 | D18 → hàng 33 trái (nút cực 3) |

## Atomic path — bấm nút → LED đổi trạng thái

1. `loop()` gọi `digitalRead(18)` — đọc latch GPIO18.
2. Không bấm: pull-up nội bộ giữ D18 @ b10 ≈ 3.3V (HIGH); cực A (h30) tách cực B (h33).
3. Bấm: nút nối cực hàng 30 ↔ hàng 33 → d33 kéo về cùng GND với g30/j30 → D18 = LOW.
4. Debounce 40ms → cạnh xuống → `ledOn = !ledOn`.
5. `digitalWrite(2, ledOn ? HIGH : LOW)` — driver GPIO2.
6. HIGH: dòng 3.3V → b5 → vàng → a27 → d27 → R → h27 → j27 → LED → j30 → cam → GND.

## Firmware ↔ hardware

| Code | GPIO | Coord | Khi chạy |
|------|------|-------|----------|
| `LED_PIN` | 2 | b5 | Output HIGH/LOW |
| `BUTTON_PIN` | 18 | b10 | INPUT_PULLUP, bấm = LOW |
| `DEBOUNCE_MS` | — | — | 40ms lọc cơ học |

## Gotchas

- **d30 ↔ g30** và **d33 ↔ g33** thông **luôn** trong nút (cùng cực). Chỉ **bấm** mới nối hàng 30 ↔ hàng 33.
- **Sai:** D18 @ hàng 30 trái + GND @ hàng 30 phải → D18 luôn gắn GND.
- **Đúng:** GND cực A (h30 phải), D18 cực B (h33 trái).
