P17 PID giữ tốc độ

Dùng encoder làm feedback để motor tự tăng/giảm PWM nhằm giữ tốc độ mục tiêu.

Cần mua thêmPIDControl loop

An toàn

Giới hạn PWM min/max trong code để tránh motor tăng tốc quá mạnh khi tham số PID chưa ổn.

Ảnh linh kiện

Linh kiện dùng thế nào

Linh kiệnVai trò
Motor + driverActuator nhận PWM.
EncoderSensor đo tốc độ thực tế.
ESP32Tính sai số và cập nhật PWM.

Các bước làm

  1. Hoàn thành P16 trước.
  2. Đặt tốc độ mục tiêu cố định.
  3. Bắt đầu với P-only, sau đó thêm I/D nếu cần.
  4. Tác động tải nhẹ vào motor và xem PWM tự bù.

Hoàn thành khi

  • Motor giữ tốc độ gần mục tiêu khi tải thay đổi nhẹ.
  • Bạn hiểu control loop: đo -> so sánh -> sửa -> lặp.
