P2 LED + điện trở

Cắm LED rời trên breadboard, blink bằng GPIO, rồi thay lần lượt 220Ω / 330Ω / 1K để so độ sáng.

Mốc 1GPIO outputOhm & dòng điện

An toàn

LED luôn phải có điện trở nối tiếp. GPIO ESP32 chỉ là 3.3V logic — không cấp 5V/VIN thẳng vào chân GPIO để điều khiển LED (chỉ HIGH/LOW ~3.3V qua code).

Điện trở đặt trước hoặc sau LED đều được — miễn là nối tiếp trong vòng D2 → … → GND.

Các bước làm

  1. Giữ LED_PIN = 2 trong src/main.cpp (tiếp P1).
  2. Cắm mạch LED + điện trở 220Ω; nối 2 jumper D2 và GND như bảng trên.
  3. Upload: pio run -t upload — LED ngoài và LED onboard nháy 1s/1s.
  4. Rút USB, thay 330Ω rồi 1K; ghi nhận độ sáng tương đối.
  5. Ghi chú vào docs/projects.md (GPIO, điện trở đã thử).

Hoàn thành khi

  • LED ngoài blink ổn định qua GPIO (không nối thẳng 3V3).
  • Đã so 220Ω / 330Ω / 1K và giải thích: điện trở lớn hơn → dòng nhỏ hơn → LED mờ hơn.
