P20 Não Python / AI

ESP32 nhận lệnh từ backend Python hoặc server AI, rồi thực thi trên robot P18/P19.

BackendHTTP/MQTT/WebSocketRobot command

An toàn

Lệnh remote phải có timeout và lệnh stop mặc định. Khi mất kết nối, robot nên dừng thay vì giữ lệnh cũ.

Ảnh linh kiện

Thành phần dùng thế nào

Thành phầnVai trò
Robot P18/P19Phần cứng thực thi lệnh.
ESP32 firmwareNhận command, validate, điều khiển motor/sensor.
Python backendRa quyết định hoặc làm API trung tâm.

Các bước làm

  1. Chọn protocol đơn giản, bắt đầu bằng HTTP JSON.
  2. ESP32 poll hoặc nhận command từ backend.
  3. Thêm heartbeat/timeout để tự stop khi mất kết nối.
  4. Nếu thêm AI, để AI quyết định lệnh cấp cao, firmware vẫn kiểm soát safety.

Hoàn thành khi

  • Backend gửi được lệnh và robot thực thi ổn định.
  • Robot dừng an toàn khi backend im lặng hoặc gửi lệnh lỗi.
