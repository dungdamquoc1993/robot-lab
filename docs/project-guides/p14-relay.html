P14 Relay

ESP32 điều khiển module relay 2 kênh 5V để nghe tiếng đóng/ngắt và hiểu cách ly tải.

RelayIsolation5V module

An toàn

Ở giai đoạn học, chỉ test tiếng relay và LED tải thấp. Không đấu điện lưới AC khi chưa có kiến thức an toàn điện.

Ảnh linh kiện

Nối dây gợi ý

Relay moduleESP32/nguồnVai trò
VCC5VNguồn coil/module.
GNDGND ESP32Mass chung.
IN1GPIO 26Điều khiển kênh 1.
IN2GPIO 27Điều khiển kênh 2 nếu cần.

Các bước làm

  1. Chỉ nối phần tín hiệu trước, upload code bật/tắt IN1 chậm.
  2. Nghe relay “tạch” khi đổi trạng thái.
  3. Kiểm tra module active LOW hay active HIGH.
  4. Nếu test tải, dùng LED/nguồn DC thấp trước.

Hoàn thành khi

  • Relay đóng/ngắt ổn định từ ESP32.
  • Bạn phân biệt được phía điều khiển GPIO và phía tiếp điểm tải.
