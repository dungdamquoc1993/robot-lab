Grid trống + chân ESP32 cố định. Mỗi project có board map riêng trong file HTML của project đó (vd P3).

65 hàng a–e ║ f–j ESP32 @ b2

Cấu trúc MB102

  [−][+]                                    [+][−]
   │ │   a  b  c  d  e  ║  f  g  h  i  j     │ │
   │ │   ●  ●  ●  ●  ●  ║  ●  ●  ●  ●  ●     │ │   ← hàng 1
   │ │   … 65 hàng vùng giữa …                │ │
   │ │                                       │ │
   └──┴── 10 cụm × (5 lỗ dọc đỏ + 5 lỗ dọc xanh)
           mỗi cụm 5 hàng nối dọc — cụm cách cụm (đứt)
  • Đỏ + · Xanh − — nối dọc trong cụm 5 lỗ; đứt giữa các cụm (và thường đứt giữa nửa trên/dưới board).
  • Vùng giữa: cùng hàng + cùng nửa (a–e hoặc f–j) → cùng cột a–e nối ngang (5 lỗ an–en).
  • Rãnh giữa e và f — a–e không nối f–j.
  • Ô xanh đậm = chân ESP32: cột b (3V3→D23) · cột j (VIN→EN). Cột i bị thân board che — dây jumper cắm cột j (cùng hàng).

ESP32 cắm vắt rãnh — cột b (trái) & j (phải), hàng 2→16

3V3 bắt đầu b2. Chân board @ b / j. Jumper tới GPIO phải → cột j (cùng hàng), không i.
