P1 Blink LED

ESP32 bật/tắt LED onboard ở GPIO 2 theo chu kỳ. Project này đã hoàn thành ngày 2026-07-04.

Trạng thái: xongGPIO 2Không cần breadboard

Linh kiện dùng thế nào

Linh kiệnVai trò
ESP32Điều khiển GPIO 2 ở mức HIGH/LOW.
LED onboardQuan sát trạng thái output mà không cần mạch ngoài.
Cáp USBNạp bằng pio run -t upload.
