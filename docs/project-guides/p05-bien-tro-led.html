P5 Biến trở → LED

Xoay biến trở 10K → ESP32 đọc ADC → map sang PWM điều độ sáng LED.

ADCPWMD34 + D2

An toàn

Biến trở: hai đầu nối 3V3 và GND, chân giữa → D34. ADC ESP32 max 3.3V — không cấp 5V.

Board map

Chi tiết: boards/p05.md. MB102 mẫu: sơ đồ chung.

Biến trở: chân 1 d20 · giữa d21 · chân 3 d22.

Jumper: đỏ a2↔a20 · cam a3↔a22 · xanh d21↔i13 · vàng a5↔a27 · cam LED a3↔i30.

Các bước làm

  1. Cắm biến trở + jumper theo board map; giữ mạch LED P4.
  2. main.cpp: chọn p05_setup / p05_loop.
  3. Upload: pio run -t upload.
  4. Serial Monitor 115200: thấy ADC=… duty=… khi xoay núm.
  5. Xoay ngược ý → đổi dây 3V3/GND hai đầu biến trở.

Hoàn thành khi

  • LED đổi độ sáng theo biến trở.
  • Phân biệt được ADC input (đọc điện áp) vs PWM output (điều sáng).
