P4 Breathing LED

Dùng PWM để LED sáng/tối mượt thay vì chỉ HIGH/LOW.

PWMLED ngoàiGPIO output

An toàn

Vẫn giữ điện trở 220Ω nối tiếp LED. PWM chỉ là bật/tắt rất nhanh theo duty cycle, không làm LED bỏ qua giới hạn dòng.

Ảnh linh kiện

Nối dây

ESP32Nối tớiGhi chú
D2 (GPIO 2)220Ω → LED anodeChân PWM; giữ mạch P2.
LED cathodeGNDJumper mass chung.

Các bước làm

  1. Dùng lại mạch LED P2.
  2. Code tăng duty cycle từ 0 lên max rồi giảm về 0.
  3. Thử tốc độ fade khác nhau để thấy hiệu ứng mượt/giật.
  4. Ghi lại GPIO, tần số PWM nếu bạn cấu hình thủ công.

Hoàn thành khi

  • LED sáng tối mượt như đang thở.
  • Bạn hiểu duty cycle là tỉ lệ thời gian ON trong một chu kỳ PWM.
