P4 Breathing LED

Dùng PWM để LED sáng/tối mượt thay vì chỉ HIGH/LOW.

PWMLED ngoàiGPIO 2

An toàn

Vẫn giữ điện trở 220Ω nối tiếp LED. PWM chỉ là bật/tắt rất nhanh theo duty cycle, không làm LED bỏ qua giới hạn dòng.

Board map — mạch LED (từ P2)

Chi tiết text: boards/p04.md. Cấu trúc MB102: sơ đồ mẫu.

Hàngb — ESP32
3GND ← cam
5D2 ← PWM vàng

Jumper: cam a3↔j30 · vàng a5↔a27.

Linh kiện: điện trở d27↔h27 · LED j27 / j30.

Các bước làm

  1. Giữ mạch LED P2 (hoặc chỉ phần LED từ board P3, bỏ jumper nâu D18).
  2. Trong src/main.cpp: comment P3, bỏ comment p04_setup / p04_loop.
  3. Upload: pio run -t upload — LED sáng tối kiểu “thở”.
  4. Thử đổi FADE_STEP_MS trong p04_breathing_led.cpp — nhỏ hơn = mượt/chậm hơn.
  5. Đối chiếu boards/p04.md.

Hoàn thành khi

  • LED sáng tối mượt như đang thở.
  • Hiểu duty cycle = tỉ lệ thời gian ON trong một chu kỳ PWM.
