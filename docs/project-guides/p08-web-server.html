P8 Web server

ESP32 tạo web server trong mạng Wi-Fi để điều khiển LED hoặc servo từ trình duyệt điện thoại.

Wi-FiHTTPIoT

An toàn

Nếu điều khiển servo, vẫn dùng nguồn 5V phù hợp và nối chung GND. Nếu chỉ điều khiển LED, dùng điện trở nối tiếp.

Ảnh linh kiện

Linh kiện dùng thế nào

Linh kiệnVai trò
ESP32Web server và controller.
LED hoặc servoThiết bị được điều khiển từ HTTP route.
Điện thoại/laptopClient mở địa chỉ IP của ESP32.

Các bước làm

  1. Kết nối ESP32 vào Wi-Fi và in IP ra Serial Monitor.
  2. Tạo route bật/tắt LED trước.
  3. Mở IP từ điện thoại cùng mạng Wi-Fi.
  4. Nếu ổn, thêm route điều khiển góc servo.

Hoàn thành khi

  • Điện thoại bật/tắt được LED hoặc đổi góc servo qua browser.
  • Bạn hiểu ESP32 đang vừa là firmware device vừa là HTTP server nhỏ.
